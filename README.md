### Updating from 2.3 to 3.0

#### Step 1: Update Podfile

  Update LPMessagingSDK Pod

	```ruby
    target ‘<YourApplicatioName>’ do
      # Update change LPMessagingSDK Pod from:
      pod 'LPMessagingSDK','~> 2.3.0'
      # To:
      pod ‘LPMessagingSDK','~> 3.0.0'
 ```
